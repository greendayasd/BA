Welcome to CrowdJump!

CrowdJump is a bachelor thesis in which you can submit your ideas to improve a simple game to your likings!

The goal of the thesis is to observe how a game is designed bottom-up with a crowd and how the decision finding
process is made.


On History you can find the changes implemented in previous iterations of the game.

If you played the game and want to submit a new feature, you should visit Ideas.

The game itself can be found here.
